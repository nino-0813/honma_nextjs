# 画像ディレクトリ構造

このディレクトリには、各ページで使用する画像を保存します。

## ディレクトリ構成

### `/home/`
ホームページで使用する画像

- **collections/** - コレクションセクションの画像（4枚）
  - コシヒカリ、亀の尾、にこまる、年間契約の各コレクション画像
  
- **parallax/** - パララックスセクションの背景画像（1枚）

### `/about/`
Aboutページで使用する画像

- **hero/** - ページ上部のヒーロー画像（1枚）
- **stances/** - 3つのスタンスセクションの画像（3枚）
  - 安心と美味しさ
  - 社会への貢献
  - 未来への継承
- **stories/** - ストーリーセクションの画像（2枚）
  - 土を磨き、命を育む
  - 「ココに生まれてよかった」そう思える島へ

### `/ambassador/`
Ambassadorページで使用する画像

- **banner/** - ページ上部のバナー画像（1枚）

### `/products/`
商品画像

- **main/** - 商品のメイン画像
- **thumbnails/** - 商品のサムネイル画像

### `/common/`
複数のページで共通して使用する画像
