# 商品メイン画像

商品のメイン画像を保存します。

## 画像の命名規則

商品のhandle（ハンドル）に基づいて命名してください。

例：
- `rice-koshihikari-300gset.jpg`
- `rice-koshihikari-5kg.jpg`
- `rice-nikomaru-5kg.jpg`

## 推奨サイズ

- アスペクト比: 1:1（正方形）
- 推奨解像度: 1200x1200px 以上
- 形式: JPG または PNG
