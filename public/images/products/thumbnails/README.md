# 商品サムネイル画像

商品のサムネイル画像（ホバー時に表示される2枚目の画像など）を保存します。

## 画像の命名規則

商品のhandle（ハンドル）に基づいて命名してください。

例：
- `rice-koshihikari-300gset-thumb.jpg`
- `rice-koshihikari-5kg-thumb.jpg`

または、メイン画像と同じファイル名で `-sub` や `-alt` などのサフィックスを追加：

例：
- `rice-koshihikari-300gset-sub.jpg`

## 推奨サイズ

- アスペクト比: 1:1（正方形）
- 推奨解像度: 1200x1200px 以上
- 形式: JPG または PNG
