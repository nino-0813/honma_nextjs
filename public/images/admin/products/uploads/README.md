# アップロード商品画像

商品管理ページでアップロードされた商品画像を保存します。

## 使用方法

1. 商品管理ページから画像をアップロード
2. 画像は自動的にこのディレクトリに保存されます
3. 保存後、商品データの `image` と `images` パスを更新してください

## ファイル命名規則

商品のhandleに基づいて命名することを推奨します。

例：
- `rice-koshihikari-5kg.jpg`
- `rice-nikomaru-10kg.png`

## 推奨サイズ

- アスペクト比: 1:1（正方形）
- 推奨解像度: 1200x1200px 以上
- 形式: JPG または PNG
