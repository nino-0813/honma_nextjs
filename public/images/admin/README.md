# 管理画面用画像ディレクトリ

商品管理ページで使用する画像を保存します。

## ディレクトリ構成

### `/products/`
商品管理で使用する商品画像

- **uploads/** - アップロードされた商品画像の保存先
  - 商品画像をアップロードすると、ここに保存されます
  - 保存後、`/images/products/main/` に移動することを推奨
  
- **temp/** - 一時保存用の画像
  - アップロード中や編集中の一時的な画像を保存
  - 不要になった画像は定期的に削除してください

### `/previews/`
プレビュー用の画像

- 商品編集時のプレビュー表示に使用
- 一時的なプレビュー画像を保存

### `/assets/`
管理画面のUI用画像

- ロゴ、アイコン、背景画像など
- 管理画面専用のアセット画像

## 使用方法

1. 商品画像をアップロードする際は、まず `/products/temp/` に保存
2. 保存が完了したら、`/products/uploads/` に移動
3. 最終的に、`/images/products/main/` または `/images/products/thumbnails/` に配置

## 注意事項

- `temp/` ディレクトリ内のファイルは定期的にクリーンアップしてください
- 本番環境では、アップロードされた画像のバリデーションを行ってください
